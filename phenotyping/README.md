# 表型参数提取模块

本目录记录基于重建点云和叶片实例分割结果的植物表型参数提取实验。当前阶段更接近工程探索：验证三维重建和分割结果是否能支持可测量的叶片级、植株级参数计算。

## 输入数据

表型数据统一放在根目录 `data/phenotypeData/`，例如：

```text
data/phenotypeData/
├── soya.ply
├── soya-leaf.ply
└── soya-handcraft.ply
```

这些文件通常是大体积 `.ply` 点云，不上传 GitHub。

## 目标参数

植株级参数：

- 株高
- 冠幅
- 点数
- 检测到的叶片/实例数量

叶片级参数：

- 叶面积
- 叶长
- 叶宽
- 叶倾角
- 叶片质心
- 主方向

## 几何后处理

报告中发现，自动实例分割结果会出现茎误检、离散点、叶片端部连接茎、多叶粘连等问题，因此设计了简单的几何后处理：

- 整簇删除明显茎状实例。
- 裁剪叶片端部细长连接茎。
- 根据相对茎轴方向对粘连叶片进行二次切分。

该流程能显著降低叶面积误差，但仍属于启发式方法，复杂粘连和遮挡缺失场景还需要人工检查或更精细的曲面重建。

## 大豆样本结果

| 指标 | 数值 |
| --- | ---: |
| GT 叶片数 | 23 |
| GT 总叶面积 | 46148.09 mm² |
| 原始自动实例数 | 29 |
| 原始结果中茎误检实例数 | 8 |
| 人工确认叶片实例数 | 21 |
| 原始总叶面积（含茎误检） | 90043.25 mm² |
| 排除茎误检后总叶面积 | 77858.76 mm² |
| 含茎误检面积相对误差 | 95.12% |
| 排除茎误检面积相对误差 | 68.72% |
| 后处理输入点数 | 77745 |
| 后处理输出点数 | 72413 |
| 删除点数 / 比例 | 5332 / 6.86% |
| 后处理实例数 | 27 |
| 后处理点云高度 | 308.86 mm |
| 后处理总投影面积 | 52770.20 mm² |
| 后处理面积误差 | 6622.11 mm² |
| 后处理面积相对误差 | 14.35% |
| 平均叶长 | 68.49 mm |
| 平均叶宽 | 36.20 mm |
| 平均单叶投影面积 | 1954.45 mm² |

结果说明：仅排除明显茎误检后，面积误差仍有 `68.72%`；加入几何后处理后，相对误差降至 `14.35%`。这说明重建和分割结果已经具备一定测量价值，但要达到稳定自动表型提取，还需要更准确的叶片实例分割和曲面面积估计。

## 后续方向

- 使用 alpha shape、网格曲面或局部曲面拟合替代简单投影面积。
- 将叶片边界、法向和曲率纳入后处理规则。
- 建立更多人工测量真值，统一评估叶长、叶宽、叶面积和株高。
- 减少人工清理依赖，形成更自动化的重建-分割-表型流程。
